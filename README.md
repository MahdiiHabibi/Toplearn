## Run Locally

Clone the project

```bash
  git clone https://github.com/MahdiiHabibi/Toplearn.git
```

Go to the project directory

```bash
  cd Toplearn
```

Install dependencies

```bash
  dotnet restore
```

```bash
  dotnet build
```

To deploy this project run

```bash
  npm run deploy
```
